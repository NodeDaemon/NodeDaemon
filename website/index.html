Quick Start

Installation

$ npm install -g @nodedaemon/core
$ yarn global add @nodedaemon/core
$ git clone https://github.com/nodedaemon/nodedaemon.git
$ cd nodedaemon
$ npm install
$ npm run build
$ npm link

Basic Usage

1. Start the daemon:

$ nodedaemon daemon -d

2. Start your application:

$ nodedaemon start app.js --name myapp --instances 4 --watch

3. Monitor processes:

$ nodedaemon list

Configuration

// nodedaemon.config.js
module.exports = {
  apps: [{
    name: 'web-server',
    script: 'server.js',
    instances: 4,
    autorestart: true,
    watch: true,
    env: {
      NODE_ENV: 'production',
      PORT: 3000
    },
    maxMemory: '1GB',
    maxRestarts: 10
  }]
};
